# 초등학생을 위한 그리기 웹앱 (Kids Drawing App)

어린이들이 다양한 주제로 그림을 그리고 창의력을 발휘할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- **추상화 그리기**: 자유롭게 선, 도형, 색을 사용하여 감정을 표현하는 페이지
- **크로키 그리기**: 제한된 시간 안에 빠르게 대상을 스케치하는 연습 공간
- **캐릭터 그리기**: 귀엽고 단순한 캐릭터를 직접 디자인할 수 있는 페이지
- **카카오 이모티콘 그리기**: 카카오 스타일의 이모티콘을 따라 그리거나 창작할 수 있는 공간
- **인물화 그리기**: 사람의 얼굴과 특징을 집중적으로 그려보는 페이지
- **풍경화 그리기**: 자연이나 건물, 거리 풍경 등을 표현하는 공간
- **6컷 만화 그리기**: 이야기 흐름이 있는 만화를 6칸으로 나눠 그릴 수 있는 페이지

## 기술 스택

- React + TypeScript
- Canvas API
- Tailwind CSS
- React Router
- Framer Motion (애니메이션)

## 개발 환경 설정

1. 저장소 클론:
```
git clone https://github.com/yourusername/kids-drawing-app.git
cd kids-drawing-app
```

2. 의존성 설치:
```
npm install
```

3. 개발 서버 실행:
```
npm run dev
```

4. 브라우저에서 `http://localhost:3000` 열기

## 사용 방법

1. 홈 화면에서 그리고 싶은 주제 선택
2. 페이지 상단의 툴바에서 색상과 붓 크기 설정
3. 캔버스에 그림 그리기
4. 저장 버튼을 클릭하여 작품 저장
